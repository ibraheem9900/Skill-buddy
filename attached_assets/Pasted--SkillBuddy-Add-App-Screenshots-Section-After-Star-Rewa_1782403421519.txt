### IN ANIMATION (when section enters — ONE SECTION AT A TIME rule applies)

```
Step 1 — Section background fades in:
  opacity: 0 → 1, duration: 300ms

Step 2 — Label + Heading slides up:
  { y: 40, opacity: 0 } → { y: 0, opacity: 1 }
  Duration: 500ms, delay: 100ms

Step 3 — iPhone POPS IN from center (main entrance):
  From: { scale: 0.3, opacity: 0, y: 60, rotateX: 20deg }
  To:   { scale: 1,   opacity: 1, y: 0,  rotateX: 0deg }
  Duration: 800ms
  Ease: cubic-bezier(0.34, 1.56, 0.64, 1)  ← this gives the "pop" overshoot feel
  Delay: 300ms

Step 4 — Phone shadow appears after phone pops in:
  Elliptical shadow below phone
  From: { opacity: 0, scaleX: 0.3 }
  To:   { opacity: 0.3, scaleX: 1 }
  Duration: 400ms, delay: 600ms

Step 5 — Subheading fades in below phone:
  { y: 20, opacity: 0 } → { y: 0, opacity: 1 }
  Duration: 400ms, delay: 800ms

Step 6 — Store badges pop in:
  Left badge: { x: -30, opacity: 0 } → { x: 0, opacity: 1 }, delay: 1000ms
  Right badge: { x: 30, opacity: 0 } → { x: 0, opacity: 1 }, delay: 1100ms
  Duration: 400ms each
```
