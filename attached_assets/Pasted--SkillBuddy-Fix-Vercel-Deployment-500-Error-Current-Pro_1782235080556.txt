### Step 4 — Fix SSR crashes (window/document errors)

Search entire codebase for any code that uses `window`, `document`, or `localStorage` OUTSIDE of:
- `useEffect` hooks
- Event handlers
- Functions that only run in browser

Any such code will crash the Vercel server. Fix by wrapping in:
```typescript
if (typeof window !== 'undefined') {
  // your code here
}
```

Or move inside `useEffect`:
```typescript
useEffect(() => {
  // window/document/localStorage code here
}, [])
```

### Step 5 — Fix i18n translation files location

Make sure ALL translation JSON files are inside the `public` folder:
```
public/
  locales/
    en/
      translation.json
    et/
      translation.json
    ru/
      translation.json
    lv/
      translation.json
    lt/
      translation.json
```

If they are anywhere else (like `src/locales/`), move them to `public/locales/`.

Update i18next config to load from:
```typescript
backend: {
  loadPath: '/locales/{{lng}}/translation.json',
}
```
