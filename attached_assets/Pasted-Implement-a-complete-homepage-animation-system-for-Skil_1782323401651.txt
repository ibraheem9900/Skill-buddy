CORE PRINCIPLE — ONE SECTION AT A TIME
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- Only ONE section is fully visible at any time
- As user scrolls DOWN: current section fades/slides out UP, next section animates IN from below
- As user scrolls UP: current section fades/slides out DOWN, previous section animates IN from above
- This is NOT a normal scrolling page — it is a FULL-SCREEN SNAP SCROLL experience
- Each section takes 100vh (full viewport height)
- Use scroll snapping: scroll-snap-type: y mandatory on the container
- Each section: scroll-snap-align: start
- When a section is active: opacity 1, transform none
- When a section is leaving (scroll down): opacity fades to 0, y moves to -80px (slides up and out)
- When a section is leaving (scroll up): opacity fades to 0, y moves to +80px (slides down and out)
- Use Framer Motion and IntersectionObserver or useScroll to detect active section
- Active section index tracked in React state
